Debug and visualization commands for ANEX C psychology modules.

## Installation

## Usage

Once installed, look for "ANEX Debug Commands" in the Extensions settings panel.

Commands will be available in chat:
- `!debug state` - View emotional state
- `!debug resources` - View resource levels
- `!debug goals` - View active goals
- More commands coming soon!
